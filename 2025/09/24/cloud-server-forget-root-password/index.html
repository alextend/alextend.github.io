cloud server forget root password

Author: Alextend Date: September 24, 2025  16:55:06

cloud server forget root password

  1. 登录控制台，并打开去服务器web版自带终端
  2. 重启服务器
  3. grub界面send Ctrl+Alt+Del(web终端一般有功能键提示)
  4. 按e进入编辑模式，找到linux/boot/,在最后加上init=”/bin/bash”
  5. 按F10或Ctrl+X重启
  6. 输入mount -o remount,rw /,回车，然后输入passwd，回车，接着输入你想修改的密码后回车，send Ctrl+Alt+Del重启服务器。
